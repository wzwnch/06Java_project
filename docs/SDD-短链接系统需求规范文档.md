# 短链接系统需求规范文档（SDD Spec）

---

## 一、业务背景与核心目标

### 1.1 业务背景

在移动互联网时代，长 URL 存在诸多问题：展示不美观、占用字符限制、难以记忆、无法追踪效果。短链接服务应运而生，将长 URL 转换为简短易记的链接，广泛应用于营销推广、短信通知、社交媒体等场景。

### 1.2 核心目标

| 目标 | 描述 |
|------|------|
| **核心功能** | 提供稳定、高效的短链接生成、跳转、管理服务 |
| **高并发支撑** | 支持海量用户注册与短链接访问，使用 Redis 替代消息队列实现异步处理 |
| **数据可观测** | 提供完整的访问统计、监控、日志能力 |
| **安全可控** | 用户敏感信息加密存储、接口脱敏返回、权限隔离 |

## 二、本期包含功能与不包含功能

### 2.1 本期包含功能

| 模块 | 功能项 |
|------|--------|
| **用户管理** | 用户注册、登录、退出、信息查询与修改、用户名检查、敏感信息脱敏、分库分表支持 |
| **短链接分组** | 分组增删改查、排序、用户上下文封装 |
| **短链接管理** | 短链接生成、跳转、修改、查询、分库分表、缓存策略（击穿/穿透/预热）、过期处理、目标网站标题/图标获取 |
| **回收站管理** | 移入回收站、恢复、移除、分页查询 |
| **短链接监控** | PV/UV/IP 统计、地区/设备/浏览器/操作系统统计、访问日志、今日/历史数据、分组统计 |

### 2.2 本期不包含功能

| 排除项 | 说明 |
|--------|------|
| 短链接自定义域名绑定 | 本期仅支持系统默认域名 |
| 短链接批量导入导出 | 后续版本迭代 |
| 多租户企业级权限 | 本期仅支持单用户维度 |
| 短链接 API 开放平台 | 后续版本迭代 |
| 短链接二维码生成 | 后续版本迭代 |

## 三、完整业务规则与限制条件

### 3.1 用户管理业务规则

| 规则编号 | 规则描述 |
|----------|----------|
| U-001 | 用户名唯一，不可重复注册 |
| U-002 | 密码必须加密存储（AES/BCrypt） |
| U-003 | 用户敏感信息（手机号、邮箱、密码）接口返回时必须脱敏 |
| U-004 | 检查用户名是否存在时，需防止缓存穿透（使用布隆过滤器或空值缓存） |
| U-005 | 海量用户注册需支持分库分表策略 |
| U-006 | 用户登录需生成 Token，退出需使 Token 失效 |
| U-007 | 用户信息修改需校验当前登录状态 |

### 3.2 短链接分组业务规则

| 规则编号 | 规则描述 |
|----------|----------|
| G-001 | 每个用户可创建多个短链接分组 |
| G-002 | 分组名称在同一用户下不可重复 |
| G-003 | 分组支持排序，排序值越小越靠前 |
| G-004 | 删除分组时，分组下的短链接需一并处理（移至回收站或拒绝删除） |
| G-005 | 用户注册成功后，系统自动创建默认分组 |

### 3.3 短链接管理业务规则

| 规则编号 | 规则描述 |
|----------|----------|
| S-001 | 短链接码生成算法需保证唯一性（如雪花算法、Base62 编码） |
| S-002 | 短链接支持设置有效期，过期后跳转指定页面 |
| S-003 | 短链接跳转需记录访问日志（PV/UV/IP 等） |
| S-004 | 短链接跳转需处理缓存击穿（分布式锁）、缓存穿透（空值缓存）、缓存预热 |
| S-005 | 短链接海量数据需分库分表存储 |
| S-006 | 短链接支持修改目标地址、有效期、分组归属 |
| S-007 | 短链接不存在时，跳转系统指定 404 页面 |
| S-008 | 创建短链接时，自动获取目标网站标题和图标 |

### 3.4 回收站管理业务规则

| 规则编号 | 规则描述 |
|----------|----------|
| R-001 | 短链接删除为逻辑删除，移入回收站 |
| R-002 | 回收站中的短链接可恢复或彻底删除 |
| R-003 | 彻底删除后数据不可恢复 |
| R-004 | 回收站支持分页查询 |

### 3.5 短链接监控业务规则

| 规则编号 | 规则描述 |
|----------|----------|
| M-001 | PV（Page View）统计每次访问请求 |
| M-002 | UV（Unique Visitor）统计独立访客（基于 Cookie/指纹） |
| M-003 | UIP（Unique IP）统计独立 IP 访问 |
| M-004 | 访问日志记录：时间、IP、地区、设备、浏览器、操作系统、网络类型 |
| M-005 | 统计数据支持按短链接、分组、时间维度查询 |
| M-006 | 高频 IP 访问需识别并记录 |
| M-007 | 访问日志异步写入（Redis 替代消息队列） |

## 四、输入条件与输出结果定义

### 4.1 用户管理接口

| 接口 | 输入条件 | 输出结果 |
|------|----------|----------|
| 用户注册 | 用户名、密码、手机号、邮箱 | 注册成功/失败、用户ID |
| 用户登录 | 用户名、密码 | Token、用户基本信息（脱敏） |
| 用户退出 | Token（Header） | 退出成功/失败 |
| 查询用户信息 | 用户ID | 用户信息（敏感字段脱敏） |
| 修改用户信息 | 用户ID、可修改字段 | 修改成功/失败 |
| 检查用户名 | 用户名 | 存在/不存在 |
| 检查手机号 | 手机号 | 存在/不存在 |

### 4.2 短链接分组接口

| 接口 | 输入条件 | 输出结果 |
|------|----------|----------|
| 新增分组 | 分组名称、排序值 | 分组ID |
| 查询分组列表 | 用户ID（Token解析） | 分组列表 |
| 修改分组 | 分组ID、新名称、新排序值 | 修改成功/失败 |
| 删除分组 | 分组ID | 删除成功/失败 |

### 4.3 短链接管理接口

| 接口 | 输入条件 | 输出结果 |
|------|----------|----------|
| 新增短链接 | 目标URL、分组ID、有效期、自定义短码（可选） | 短链接码、完整短链接 |
| 查询短链接列表 | 分组ID、分页参数 | 短链接列表（分页） |
| 修改短链接 | 短链接码、可修改字段 | 修改成功/失败 |
| 删除短链接 | 短链接码 | 删除成功/失败（移入回收站） |
| 短链接跳转 | 短链接码 | 302 重定向至目标URL 或 404 页面 |

### 4.4 回收站接口

| 接口 | 输入条件 | 输出结果 |
|------|----------|----------|
| 查询回收站 | 分页参数 | 回收站短链接列表（分页） |
| 恢复短链接 | 短链接码 | 恢复成功/失败 |
| 彻底删除 | 短链接码 | 删除成功/失败 |

### 4.5 短链接监控接口

| 接口 | 输入条件 | 输出结果 |
|------|----------|----------|
| 单链接统计 | 短链接码 | PV/UV/UIP 汇总数据 |
| 访问日志查询 | 短链接码、时间范围、分页参数 | 访问日志列表（分页） |
| 今日统计 | 用户ID/分组ID | 今日 PV/UV/UIP |
| 历史统计 | 用户ID/分组ID、时间范围 | 历史 PV/UV/UIP 趋势数据 |
| 分组统计 | 分组ID | 分组下所有链接汇总数据 |
| 高频 IP 统计 | 短链接码、时间范围 | 高频 IP 列表 |

## 五、边界场景与异常情况处理

### 5.1 用户管理边界场景

| 场景 | 处理方式 |
|------|----------|
| 用户名已存在 | 返回错误码，提示"用户名已被注册" |
| 用户名/密码格式不合法 | 返回错误码，提示格式要求 |
| 手机号/邮箱已绑定其他账号 | 返回错误码，提示"手机号/邮箱已被使用" |
| 登录密码错误 | 返回错误码，连续错误 N 次锁定账号 |
| Token 过期/无效 | 返回 401，提示重新登录 |
| 缓存穿透（查询不存在的用户名） | 布隆过滤器拦截 或 空值缓存短时间 |

### 5.2 短链接管理边界场景

| 场景 | 处理方式 |
|------|----------|
| 目标 URL 格式不合法 | 返回错误码，提示 URL 格式要求 |
| 短链接码重复（极低概率） | 重新生成短链接码 |
| 短链接已过期 | 跳转系统过期提示页 |
| 短链接不存在 | 跳转系统 404 页面 |
| 目标网站无法访问 | 记录错误日志，返回默认标题/图标 |
| 缓存击穿（热点短链接过期瞬间） | 分布式锁，只允许一个请求重建缓存 |
| 缓存穿透（查询不存在的短链接） | 空值缓存，设置短过期时间 |
| 分组不存在 | 返回错误码，提示分组无效 |

### 5.3 回收站边界场景

| 场景 | 处理方式 |
|------|----------|
| 恢复时短链接码已被占用 | 返回错误码，提示冲突 |
| 彻底删除后再次操作 | 返回错误码，提示短链接不存在 |
| 回收站为空 | 返回空列表 |

### 5.4 监控统计边界场景

| 场景 | 处理方式 |
|------|----------|
| 无访问数据 | 返回空数据或零值 |
| 时间范围超过限制 | 返回错误码，提示最大查询范围 |
| Redis 队列积压 | 后台异步消费，保证最终一致性 |
| IP 解析失败 | 记录原始 IP，地区标记为"未知" |

### 5.5 系统级异常处理

| 异常类型 | 处理方式 |
|----------|----------|
| 数据库连接失败 | 返回 500，记录错误日志，告警通知 |
| Redis 连接失败 | 降级处理，直接查询数据库 |
| 参数校验失败 | 返回 400，提示具体参数错误 |
| 未登录访问受保护接口 | 返回 401，提示需要登录 |
| 无权限访问 | 返回 403，提示权限不足 |
| 系统内部异常 | 返回 500，统一错误码，隐藏堆栈信息 |

## 六、验收标准

### 6.1 功能验收标准

| 模块 | 验收标准 |
|------|----------|
| **用户注册** | 正常注册成功；重复用户名注册失败；敏感信息加密存储；注册后自动创建默认分组 |
| **用户登录** | 正确凭证登录成功并返回 Token；错误凭证登录失败；Token 有效期内可访问受保护接口 |
| **用户退出** | 退出后 Token 立即失效 |
| **用户名检查** | 存在的用户名返回存在；不存在的用户名返回不存在；无缓存穿透问题 |
| **用户信息查询/修改** | 敏感字段脱敏返回；修改后数据即时生效 |

| 模块 | 验收标准 |
|------|----------|
| **分组创建** | 正常创建成功；重复名称创建失败 |
| **分组查询** | 返回当前用户全部分组；按排序值排序 |
| **分组修改** | 正常修改成功；修改排序值后列表顺序正确 |
| **分组删除** | 删除成功；分组下有短链接时按规则处理 |

| 模块 | 验收标准 |
|------|----------|
| **短链接创建** | 正常生成短链接；目标 URL 合法性校验；自动获取标题和图标 |
| **短链接跳转** | 正常跳转至目标 URL；过期短链接跳转过期页；不存在短链接跳转 404 页 |
| **短链接查询** | 分页正确；按条件筛选正确 |
| **短链接修改** | 正常修改成功；修改后跳转地址更新 |
| **缓存策略** | 热点短链接无缓存击穿；不存在的短链接无缓存穿透；系统启动后缓存预热 |

| 模块 | 验收标准 |
|------|----------|
| **回收站查询** | 返回已删除短链接列表；分页正确 |
| **回收站恢复** | 恢复后短链接可正常跳转 |
| **回收站删除** | 彻底删除后数据不可恢复 |

| 模块 | 验收标准 |
|------|----------|
| **PV/UV/UIP 统计** | 数据准确；访问一次 PV+1；同一访客 UV 不重复 |
| **访问日志** | 日志记录完整（时间、IP、地区、设备等）；分页查询正确 |
| **今日/历史统计** | 数据准确；时间范围筛选正确 |
| **分组统计** | 分组下所有链接数据汇总正确 |
| **高频 IP** | 正确识别高频访问 IP |

### 6.2 性能验收标准

| 指标 | 标准 |
|------|------|
| 短链接跳转响应时间 | P99 < 100ms（缓存命中） |
| 短链接创建响应时间 | P99 < 500ms |
| 用户注册响应时间 | P99 < 500ms |
| 并发短链接跳转 | 支持 10000 QPS |
| 统计数据查询响应时间 | P99 < 1s |

### 6.3 安全验收标准

| 指标 | 标准 |
|------|------|
| 密码存储 | 加密存储，不可逆 |
| 敏感信息传输 | HTTPS 加密 |
| 敏感信息返回 | 脱敏处理（手机号、邮箱） |
| Token 安全 | 有效期控制；退出即失效 |
| SQL 注入 | 所有参数校验，使用预编译 |
| XSS 攻击 | 输出转义 |

### 6.4 可用性验收标准

| 指标 | 标准 |
|------|------|
| 系统可用性 | ≥ 99.9% |
| 数据一致性 | 最终一致性，统计数据延迟 < 5min |
| 容错能力 | Redis 故障时降级可用 |
